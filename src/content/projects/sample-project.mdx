## Architecture

The project follows a clean architecture pattern:

1. **Components**: Reusable UI components
2. **Routes**: Page-level components with routing logic
3. **Utils**: Helper functions and utilities
4. **Types**: TypeScript type definitions

## Challenges & Solutions

### Challenge 1: State Management
We needed a lightweight state management solution that wouldn't add unnecessary complexity.

**Solution**: Used React's built-in `useState` and `useContext` for simple state management, avoiding the overhead of external libraries.

### Challenge 2: Performance Optimization
Initial load times were slower than desired.

**Solution**: Implemented code splitting with TanStack Router's built-in lazy loading and optimized bundle sizes.
