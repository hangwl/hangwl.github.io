# Figma for Developers

This guide is designed for developers transitioning into design work or looking to improve their wireframing skills. It focuses on the concepts that map directly to coding (like Auto Layout -> Flexbox).

## Documentation Index

- **[Figma Interface Basics](/notes/figma-basics)**: Understanding the UI, shortcuts, and the difference between Groups and Frames.
- **[Auto Layout (Flexbox)](/notes/auto-layout)**: Mastering Flexbox in Figma for responsive designs.
- **[Components & Variants](/notes/components)**: Creating reusable components and managing their states (props).
- **[Wireframing Workflow](/notes/wireframing)**: Best practices for Lo-Fi wireframing and prototyping flows.
- **[Developer Mode](/notes/developer-mode)**: Inspecting designs and exporting assets for implementation.
- **[Material UI Toolkit](./mui/intro)**: Using the official MUI design kit to sync design and code tokens.
