## Auto Layout (Flexbox)

Auto Layout is Figma's version of Flexbox. It is essential for making designs that actually look like the final product.

- **Enable**: Select a frame and press `Shift + A`.
- **Direction**: Row (horizontal) or Column (vertical).
- **Resizing**:
  - **Fixed**: Explicit width/height (e.g., `width: 200px`).
  - **Hug**: `width: max-content` (Shrinks to fit children).
  - **Fill**: `flex: 1` (Grows to fill available space).
- **Padding/Gap**: Control internal spacing just like CSS.

Frames with Auto Layout act like flex containers, allowing for responsive behavior within your designs.
