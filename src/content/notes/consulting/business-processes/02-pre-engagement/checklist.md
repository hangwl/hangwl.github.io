# Pre-Engagement Checklist

## Overview

This checklist ensures all necessary steps are completed before an engagement formally begins. Use this as a guide from opportunity identification through contract signing.

---

## Phase 1: Opportunity Qualification

### Client Assessment
- [ ] Initial client meeting completed
- [ ] Client needs and pain points documented
- [ ] Budget range confirmed or estimated
- [ ] Decision-making process understood
- [ ] Timeline and urgency clarified
- [ ] Competitive landscape assessed

### Internal Assessment
- [ ] Opportunity registered in CRM (CRM)
- [ ] Pursuit team identified
- [ ] Resource availability confirmed
- [ ] Capability fit assessed
- [ ] Go/No-Go decision documented

---

## Phase 2: Independence and Risk

### Independence Checks (MANDATORY)
- [ ] Independence check initiated in GIMS (Global Independence Monitoring System)
- [ ] Client and related entities checked
- [ ] Business relationships reviewed
- [ ] Prior Firm relationships identified
- [ ] Independence approval obtained

### Risk Assessment
- [ ] Client acceptance procedures completed
- [ ] Anti-money laundering (AML) checks done
- [ ] Sanctions screening completed
- [ ] Reputation risk assessment completed
- [ ] Engagement risk classification determined

### Conflicts Check
- [ ] Conflict check submitted
- [ ] Potential conflicts identified and addressed
- [ ] Conflict clearance obtained

---

## Phase 3: Proposal Development

### Requirements and Scoping
- [ ] Client requirements documented
- [ ] Solution approach defined
- [ ] Scope boundaries clearly stated
- [ ] Assumptions documented
- [ ] Exclusions listed
- [ ] Dependencies identified

### Solution Design
- [ ] Technical approach developed
- [ ] Methodology selected (Agile, Waterfall, Hybrid)
- [ ] Tools and technologies identified
- [ ] Team structure designed
- [ ] Timeline and phases defined
- [ ] Deliverables specified

### Commercial
- [ ] Effort estimation completed
- [ ] Pricing model selected (Fixed Fee, T&M, Outcome-based)
- [ ] Rate card applied
- [ ] Margin calculation done
- [ ] Pricing approval obtained
- [ ] Payment terms defined

### Proposal Document
- [ ] Proposal template selected
- [ ] Executive summary drafted
- [ ] Solution narrative completed
- [ ] Team profiles included
- [ ] Relevant experience/case studies added
- [ ] Internal review completed
- [ ] Quality review (if required) completed
- [ ] Partner approval obtained

---

## Phase 4: Client Engagement

### Proposal Presentation
- [ ] Presentation materials prepared
- [ ] Presentation team briefed
- [ ] Q&A anticipated and prepared
- [ ] Proposal submitted/presented

### Negotiation
- [ ] Client feedback received and documented
- [ ] Scope/pricing adjustments made (if needed)
- [ ] Updated proposal submitted (if applicable)
- [ ] Final terms agreed

---

## Phase 5: Contracting

### Contract Documentation
- [ ] Engagement letter / SOW drafted
- [ ] Terms and conditions reviewed
- [ ] Liability and insurance requirements confirmed
- [ ] IP ownership terms defined
- [ ] Confidentiality provisions included
- [ ] Data protection requirements addressed

### Internal Approvals
- [ ] Legal review completed
- [ ] Finance review completed
- [ ] Risk management approval (if high risk)
- [ ] Partner approval obtained
- [ ] Signing authority confirmed

### Contract Execution
- [ ] Contract sent to client
- [ ] Client signatures obtained
- [ ] Firm signatures obtained
- [ ] Fully executed contract filed
- [ ] Contract registered in systems

---

## Phase 6: Pre-Mobilization

### Team Preparation
- [ ] Team members confirmed
- [ ] Resource bookings made
- [ ] Team briefing scheduled
- [ ] Onboarding materials prepared

### Client Logistics
- [ ] Client access requirements identified
- [ ] Background check requirements (if any)
- [ ] Badge/building access requested
- [ ] System access requirements listed
- [ ] Client contact list obtained

### Project Setup
- [ ] Project code created
- [ ] Time booking codes set up
- [ ] Expense codes configured
- [ ] Project folder structure created
- [ ] Communication channels established
