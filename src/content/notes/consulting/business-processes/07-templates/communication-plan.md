## 6. Communication Channels

| Channel | Use For | Guidelines |
|---------|---------|------------|
| **Email** | Formal communications, documents, decisions | Use for records, include relevant parties |
| **Teams/Slack** | Day-to-day collaboration, quick questions | Use appropriate channels, keep professional |
| **Phone/Video** | Urgent matters, complex discussions | Schedule when possible, summarize in writing |
| **SharePoint** | Document sharing, collaboration | Follow naming conventions, version control |
| **Meetings** | Scheduled discussions, reviews, decisions | Prepare agenda, document outcomes |

---

## 7. Escalation Path

### 7.1 Issue Escalation

```
Level 1: Project Team → Project Manager
Level 2: Project Manager → Engagement Manager
Level 3: Engagement Manager → Engagement Partner + Client Sponsor
Level 4: Engagement Partner → Steering Committee
```

### 7.2 Escalation Triggers

- Unresolved blockers >48 hours
- Schedule delay >1 week
- Budget variance >10%
- Resource unavailability
- Stakeholder alignment issues
