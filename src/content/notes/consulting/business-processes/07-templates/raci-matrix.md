## RACI Definition

| Letter | Role | Description |
|--------|------|-------------|
| **R** | Responsible | Does the work; can be multiple people |
| **A** | Accountable | Final decision authority; only one per activity |
| **C** | Consulted | Provides input before decision/action |
| **I** | Informed | Notified after decision/action |

## Notes

- "R" can be multiple people
- If no "C" or "I", leave blank (not all need to be filled)
- Review and update as team changes
- Ensure all parties understand their responsibilities
